# Lesson 6: More RDKit Descriptors

Today we added more real RDKit descriptors.

## Why This Matters

SwissADME gives many properties.

To become more useful, our platform also needs a richer molecule profile.

## New Descriptors

Molecular formula:

```text
C9H8O4
```

This tells the exact atom composition.

Exact mass:

This is calculated from exact isotope masses. It is useful in analytical chemistry and mass spectrometry.

Heavy atom count:

Number of non-hydrogen atoms.

Ring count:

Number of ring systems in the molecule.

Aromatic rings:

Number of aromatic rings, like benzene-like systems.

Fraction Csp3:

This gives an idea of molecular three-dimensional character.

Formal charge:

Overall formal charge of the molecule.

Molar refractivity:

A descriptor related to size and polarizability.

## Company Meaning

This is still not full ADMET.

But it is a stronger descriptor foundation. ADMET models use descriptors like these as inputs.
